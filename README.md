# 名称（ClockPlus）  
ディスクトップに、アナログ時計・デジタル時計を表示するソフトです。


# 動作画面  
![Image](https://github.com/user-attachments/assets/3868f8f2-8dc8-4e73-b0a3-10a57909579d)  

# 特徴    
表示のデザインを自由にカスタマイズ出来るのが特徴です。
その他の機能として開発当初、アラーム・タイマー・ストップウォッチの機能を有していたのですが、
当方プログラミングの知識がまだ浅いため不具合が多く、そのままでは公開できないと判断したため、
時計表示機能のみを残し、公開するに至りました。
また、Windows起動時に実行させる機能もうまく動作しなかったので無効に致しました。
将来的には随時、この機能を含めたバージョンを公開していければ良いなと考えています。

※起動時にディスクトップ画面が一瞬チラつきますがこれは仕様です。
  
# 動作条件  
・Windows11 64Bit  
・.NET8.0以上
  
# インストール方法  
最新のClockPlus.zip及びskin.zipを、https://github.com/HosoQchan/ClockPlus/releases からダウンロードし、
ClockPlus.zipを適当な場所に解凍して下さい。  
解凍してできたClockPlusディレクトリ下の"ClockPlus.exe"をダブルクリックで起動出来ます。  

skin.zipには、おまけskinとして2種類のskinが入っています。
skin.zipを解凍すると「Alarm」「Neon」の2つのフォルダが出来ますので、
ClockPlusを実行後に作成される、「C:\Users\ユーザー名\AppData\Local\ClockPlus\skin」フォルダの中に、
この2つのフォルダをコピーしてお使いください。
skinの適用は、アナログ時計の設定画面から行えます。

# 使い方  
基本的な操作としては、時計をD&Dで移動、右クリックの「設定」で表示の設定、「閉じる」で時計を非表示です。
再度時計を表示するには、タスクバー上のClockPlusアイコンを右クリックして「基本設定」画面から行って下さい。  
<img width="189" alt="Image" src="https://github.com/user-attachments/assets/f4c97c72-83b9-49db-94a1-08baa58186a8" />  
![Image](https://github.com/user-attachments/assets/adc62ae8-6b19-476c-8d43-10e265baec20)  
また、Windows起動時に実行させたい場合、現状お手数ですが手動にてスタートアップに登録してください。  

このソフトは起動するとタスクバーに常駐します。
ソフトを終了させるには、タスクバー上のClockPlusアイコンを右クリックして「終了」を選んでください。

※何かしらの原因で、時計の位置がおかしくなったり表示されなくなった場合は、「基本設定」から各時計表示の設定に移って、
「初期値」ボタンをクリックしてみてください。
